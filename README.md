# 신승민 기술 블로그

Next.js static export 기반 개인 기술 블로그입니다.

## 실행

```bash
npm install
npm run dev
```

로컬 주소:

```text
http://127.0.0.1:3000
```

## 빌드

```bash
npm run build
```

정적 산출물은 `out/`에 생성됩니다.

## 글 작성

Markdown 글은 `content/posts`에 추가합니다.
